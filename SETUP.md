# Setup Instructions

## Create GitHub Repository

1. Go to https://github.com/new
2. Repository name: `mdde-semantic`
3. Description: `MDDE Semantic Module - Semantic contracts, metrics, and decision traces`
4. Visibility: **Private**
5. Click "Create repository"

Then push:

```bash
cd c:/Repos/mdde-semantic
git remote add origin https://github.com/jacovanderlaan/mdde-semantic.git
git push -u origin main
```
